Detailed Description

A script which slows everything down.

We do so using time.TimeScale and not GameManager.gamePlaySpeed because we want to have the effect of slowing down everything by the same factor.

We also make each next such Floor destroyed slow down the time less: Time.timeScale *= 1 - (slowDownRate / numberOfFloorsDestroyed)

override void Destroy ( )
virtual

Holds information about what happents when the Floor is destroyed, can be called from other scripts.

We set that we have destroyed the Floor and invoke the base Destroy method after a given time.
We decrease the Time.timeScale by the appropriate factor.
Depending on the numberOfFloorsDestroyed we decrease by a different factor.
Time.timeScale *= 1 - (slowDownRate / numberOfFloorsDestroyed)
i.e. if we have a slowDownRate equal to 0.5f and 1 active TimeSlowerFloorScript's and we Destroy another one, we will decrease the Time.timeScale by a factor of 1 - (0.5f / 2) = 1 - 0.25f = 0.75f. And therefore the overal Time.timeScale will be 0.5f * 0.75f = 0.375f.

void Desactivate ( )

Desactivate this instance.

We call Floor.Destroy and decrease the numberOfFloorsDestroyed We revert the slowing of time that was made in the Destroy method with the same logic.
And we reactivate all its components, although the whole GameObject itself is descativated by Floor.Destroy. We do this so that when we "respawn" the same Floor, it will be visible and work as supposed.
