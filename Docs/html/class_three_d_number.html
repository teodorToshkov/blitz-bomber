Detailed Description

This script controls a 3D text which represents a number.

ThreeDNumber.cs

It stores the GameObjects which represent the digits. It controls the score we have in the gameplay.

void Update ( )
private

We have the static number which can be manipulated through other scripts and we have to see if the number we have displayed differs or not.

If it differs, we destroy all digits that are displayed currently and add the appropriate ones.
We loop throuhg each digit in the "_number" using a buffer, which we get the last digit of and devide by 10 to go to the next one until the number we are left with becomes 0.
We get the GameObject of the appropriate digit (digits [bugffer % 10]) and istantiate it at the position which is equal to the "offset" we have set and make its parent our number so that we may delete it easily when we need to update our number again.
